# 1. 結論（サマリー）

見積書は、仕様書の要求内容に応じて **工程単位・設備単位・配管単位・材料明細単位** で積算されており、
算出ロジックは次のように分類できます：

### ✔ 見積ロジック大分類

| 区分                    | ロジック概要                                       |
| --------------------- | -------------------------------------------- |
| **① 工種別「一式」金額算出**     | 仕様書で細目指定されている場合、工種単位で「一式」計上（基礎・電気・ガス・衛生・解体等） |
| **② 材料費 × 数量**        | ケーブル、照明器具、配管、分電盤、キュービクルなどは数量明細＋単価で算出         |
| **③ 労務費・施工費（工費）**     | 配線工事、配管工事、掘削、埋設、撤去、盛替え工事など「工費」計上             |
| **④ 間接費（共通仮設・現場管理費）** | 多くが一式に内包（詳細は明示されず）                           |
| **⑤ 法定福利費・諸経費**       | 仕様書の別途規定により工費の一定率で算出（例：16.07%）               |

特に、**電気設備見積書（約2億円規模）** は、
仕様書に記載されている以下の項目が大量に含まれるため **材料明細＋工費積算方式** が中心。

* 高圧受変電設備
* キュービクル設置
* 配線・幹線工事
* LAN/放送/火災報知/電話
* LED照明、大量の照明器具
* 外灯
* 既存体育館との接続工事（大規模）

都市ガスの見積書（約1340万円）は、基本的に

* 配管材
* 専用部材
* 掘削・埋設・復旧
* 解体工事
* 法定福利費（16.07%）

の積算で構成されています。

## 🔍 電気設備見積の計算ロジック（分類ごと）

| 要素                  | 計算方法                                 |
| ------------------- | ------------------------------------ |
| **機器（キュービクル・分電盤等）** | メーカー見積ベース × 1台数量                     |
| **ケーブル**            | （単価 × m数量）×種別（CVT60sq、VE管等）          |
| **配管・配線工事**         | 労務費：職人×日数＋施工手間 ×（管径・距離）              |
| **照明器具**            | 器具単価 × 灯数（LED：391灯など）                |
| **スイッチ・コンセント**      | 単価 × 個数                              |
| **屋外設備工事**          | 掘削・埋設・復旧は「1式」計上                      |
| **解体・撤去作業**         | 人工 × 日数で算出（1,800,000円/式）             |
| **法定福利費（p.12）**     | 電気設備工事費 5,430,000円 × 16.07%＝872,601円 |

# 4. 全見積共通：積算ロジックの一般式

見積書から逆算したロジックは以下の通り：

---

## ① 材料（機器）費

**材料単価 × 数量**
例：
LED照明（20,700円 × 10灯 ＝ 207,000円）
（p.8 電気設備）

---

## ② 施工費・労務費

**作業員単価 × 人数 × 日数**
または
**施工1式（施工範囲が広い場合）**

---

## ③ 掘削・埋戻し・復旧

舗装復旧＝
**（㎡単価 × 面積）**
または
**舗装復旧1式**

都市ガスPDF p.5–6 に多数の復旧項目あり。

---

## ④ 間接費

見積書に明示されていないが一般的に次を含む：

* 現場管理費
* 共通仮設費（仮囲い・トイレなど）
* 安全管理費

多くは「工事費一式」に内包。

---

## ⑤ 法定福利費（明示あり）

電気設備：
**5,430,000 × 16.07% = 872,601円（p.12）**

都市ガス設備：
**A × 16.07% = 657,263円（p.8）**
